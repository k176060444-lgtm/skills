### 百度热搜数据获取
- baidu-search 技能的 API 有延迟，返回数据滞后约4天，不是实时数据
- 获取实时热搜正确方法：直接 curl 爬 `https://top.baidu.com/board?platform=pc&tab=realtime`
- 热搜推送格式：圆角卡片式，左侧头像右侧内容，每条热搜链接必须完整URL编码
